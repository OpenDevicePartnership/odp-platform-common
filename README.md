# odp-platform-common

Common tools, libraries, and test utilities shared across Open Device Partnership platforms.
See [ODP Documentation](https://opendevicepartnership.github.io/documentation/guide/overview.html) for more details.

## Components

- **ec-test-lib/** — Rust library providing EC transport traits and implementations
- **ec-test-tui/** — Ratatui-based TUI for demoing EC features
- **ec-test-win/** — Windows native components
